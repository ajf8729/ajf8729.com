Part V: ConfigMgr - Configuration

  1. Configure ConfigMgr site
    • https://github.com/ajf8729/Lab/blob/main/04-ConfigureCM.ps1
  2. Create OU-based collections
    • https://github.com/ajf8729/Lab/blob/main/05-CreateOUCollectionStructure.ps1
  3. Install CM client on LABCM01
    • "F:\Program Files\Microsoft Configuration Manager\Client\ccmsetup.exe"
  4. Configure Client Settings
    • New-CMClientSetting -Name "LAB Default Client Settings" -Type Device
    • New-CMClientSettingDeployment -Name "LAB Default Client Settings" -CollectionName "All Systems"
    • Edit “LAB Default Client Settings”
      • Computer Agent
        • Organization name
        • PowerShell execution policy
      • Software Center
        • Company name
        • Color scheme
        • Foreground color
        • Disable “Hide installed applications”
        • Enable “Hide Application Catalog”
      • Software Updates
        • Enable software updates
        • Enable third party software updates
  5. Install/Configure Software Update Point
    • https://github.com/ajf8729/Lab/blob/main/06-InstallSoftwareUpdatePoint.ps1
  6. Install/Configure State Migration Point
