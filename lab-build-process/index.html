Lab Build Process

Initial VM Setup#

  1. Build VMs
    • https://github.com/ajf8729/Lab-Stuff/blob/main/Build/01-BuildVMs.ps1
  2. Start VMs and install Windows Server 2022 Datacenter GUI
  3. Set Administrator passwords and log in to VMs
  4. Configure hostnames
    • LABDC01A: Rename-Computer -NewName LABDC01A
    • LABCM01: Rename-Computer -NewName LABCM01
  5. Configure IP addresses
    • LABDC01A: New-NetIPAddress -InterfaceAlias Ethernet -AddressFamily IPv4 -IPAddress 172.20.1.51 -PrefixLength 24 -DefaultGateway 172.20.1.1
    • LABCM01: New-NetIPAddress -InterfaceAlias Ethernet -AddressFamily IPv4 -IPAddress 172.20.1.52 -PrefixLength 24 -DefaultGateway 172.20.1.1
  6. Configure DNS servers
    • Set-DnsClientServerAddress -InterfaceAlias Ethernet -ServerAddresses @("172.20.1.13","172.20.1.19")
  7. Configure time zone
    • Set-TimeZone -Id "Eastern Standard Time"
  8. Configure PSGallery & install PSWindowsUpdate
    • Install-PackageProvider -Name NuGet -Force
    • Register-PSRepository -Default
    • Set-PSRepository -Name PSGallery -InstallationPolicy Trusted
    • Install-Module -Name PSWindowsUpdate
  9. Install Windows Updates
    • Import-Module -Name PSWindowsUpdate
    • Get-WindowsUpdate -AcceptAll -Install -Verbose

Active Directory - Installation & Configuration#

  1. Install Active Directory
    • https://github.com/ajf8729/Lab-Stuff/blob/main/Build/02-CreateAD.ps1
  2. Configure Active Directory
    • https://github.com/ajf8729/Lab-Stuff/blob/main/Build/03-ConfigureAD.ps1
  3. Configure GPOs
    • Server - ConfigMgr
      • User Rights Assignments
        • “Lock pages in memory”: LAB\svc_CM_SQL$
        • “Log on as a service”: LAB\svc_CM_SQL$
      • Windows Firewall -> New inbound rule
        • Custom
        • All Programs
        • Local TCP ports 1433, 4022
        • Any:Any, Allow, All profiles
        • Name: “LAB - ConfigMgr - SQL”
    • Server - Default Security Policy -> New local group preference
      • Add LocalAdmin_Servers to Administrators
      • Set group description to “GPO-Enforced Group”
      • Enable “Delete all member users” and “Delete all member groups”
    • Workstation - Default Security Policy -> New local group preference
      • Add LocalAdmin_Workstations to Administrators
      • Set group description to “GPO-Enforced Group”
      • Enable “Delete all member users” and “Delete all member groups”

ConfigMgr - SQL Installation#

  1. Update DNS servers
    • Set-DnsClientServerAddress -InterfaceAlias Ethernet -ServerAddresses @("172.20.1.51")
  2. Domain join
    • Add-Computer -DomainName "lab.ajf8729.com" -Restart
  3. Configure disks
    • Get-Disk | ?{$_.OperationalStatus -eq "Offline"} | Set-Disk -IsOffline $false
    • Get-Disk | ?{$_.IsReadOnly -eq $true} | Initialize-Disk -PartitionStyle GPT
    • New-Volume -DiskNumber 1 -FileSystem NTFS -DriveLetter "E" -FriendlyName "SQL" | Out-Null
    • New-Volume -DiskNumber 2 -FileSystem NTFS -DriveLetter "F" -FriendlyName "CM" | Out-Null
    • New-Item -Path "F:\" -Name "SOURCE" -ItemType Directory | Out-Null
  4. Install RSAT tools
    • Install-WindowsFeature -Name RSAT-AD-PowerShell,RSAT-AD-AdminCenter,RSAT-ADDS-Tools,GPMC
  5. Install gMSA
    • Install-ADServiceAccount -Identity "svc_CM_SQL"
    • Restart-Computer
  6. Stage media
    • Windows ADK
      • https://docs.microsoft.com/windows-hardware/get-started/adk-install#download-the-adk-for-windows-11
      • adksetup.exe /layout F:\SOURCE\ADK\ADK
      • adkwinpesetup.exe /layout F:\SOURCE\ADK\WinPE
    • ConfigMgr Technical Preview (2106 as of 2021-08-29)
      • https://www.microsoft.com/evalcenter/evaluate-microsoft-endpoint-configuration-manager-technical-preview/
      • https://download.microsoft.com/download/D/8/E/D8E795CE-44D7-40B7-9067-D3D1313865E5/Configmgr_TechPreview2106.exe
    • SQL Server 2019 Media
      • https://www.microsoft.com/sql-server/sql-server-downloads
    • SQL Server 2019 installation configuration file
      • https://github.com/ajf8729/Lab-Stuff/blob/main/Build/SQLConfig.ini
    • SQL Server 2019 LCU (CU 12 as of 2021-08-29)
      • https://www.microsoft.com/en-us/download/details.aspx?id=100809
    • SQL Server Management Studio (18.9.2 as of 2021-08-29)
      • https://aka.ms/ssmsfullsetup
  7. Install SQL Server 2019
    • setup.exe /IAcceptSQLServerLicenseTerms /ConfigurationFile="F:\SOURCE\SQLConfig.ini"
  8. Install SQL Server 2019 LCU
    • SQLServer2019-KB5004524-x64.exe /qs /IAcceptSQLServerLicenseTerms /Action=Patch /AllInstances
  9. Install SQL Server Management Studio
    • SSMS-Setup-ENU.exe /install /passive
  10. Restart LABCM01
  11. Set SQL maximum memory usage to 1 GB & restart service

ConfigMgr - Prerequisites & Site Installation#

  1. Create NO_SMS_ON_DRIVE.SMS files
    • New-Item -Path 'C:\' -Name "NO_SMS_ON_DRIVE.SMS" -ItemType File | Out-Null
    • New-Item -Path 'E:\' -Name "NO_SMS_ON_DRIVE.SMS" -ItemType File | Out-Null
  2. Install Windows features for Primary Site, Management Point, Distribution Point, State Migration Point, and Software Update Point roles
    • Install-WindowsFeature -Name (Get-Content -Path "WindowsFeatures.txt")
  3. Extend AD schema
    • Copy ConfigMgr_ad_schema.ldf to DC
    • Edit ConfigMgr_ad_schema.ldf
      • Find/Replace all DC=x with DC=lab,DC=ajf8729,DC=com
    • ldifde.exe -i -f ConfigMgr_ad_schema.ldf -v
  4. WSUS Postinstall
    • wsusutil.exe postinstall SQL_INSTANCE_NAME="labcm01.lab.ajf8729.com" CONTENT_DIR="F:\WSUS"
  5. Install Windows ADK
    • adksetup.exe /features OptionId.DeploymentTools OptionId.ImagingAndConfigurationDesigner OptionId.ICDConfigurationDesigner OptionId.UserStateMigrationTool /ceip on
    • adkwinpesetup.exe /features OptionId.WindowsPreinstallationEnvironment /ceip on
  6. Download updated setup files
    • \Configmgr_TechPreview2106\SMSSETUP\BIN\X64\setupdl.exe "F:\SOURCE\Downloads"
  7. Install ConfigMgr primary site
    • setup.exe /script ConfigMgr.ini
