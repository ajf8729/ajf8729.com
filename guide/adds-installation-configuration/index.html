  4. Configure GPOs
    • Server - ConfigMgr
      • User Rights Assignments
        • “Lock pages in memory”: LAB\svc_CM_SQL$
        • “Log on as a service”: LAB\svc_CM_SQL$
      • Windows Firewall -> New inbound rule
        • Custom
        • All Programs
        • Local TCP ports 1433, 4022
        • Source: ConfigMgr Server IP
        • Destination: ConfigMgr Server IP
        • Allow
        • All profiles
        • Name: “ConfigMgr - SQL Traffic”
    • Server - Default Security Policy -> New local group preference
      • Add LocalAdmin_Servers to Administrators
      • Set group description to “GPO-Enforced Group”
      • Enable “Delete all member users” and “Delete all member groups”
    • Workstation - Default Security Policy -> New local group preference
      • Add LocalAdmin_Workstations to Administrators
      • Set group description to “GPO-Enforced Group”
      • Enable “Delete all member users” and “Delete all member groups”
