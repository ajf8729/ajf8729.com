OneDrive for Business, Known Folder Move, and Enterprise State Roaming

  1. Configure OneDrive for Business (ODfB) and Known Folder Move (KFM)
    • https://docs.microsoft.com/en-us/onedrive/use-group-policy
    • https://docs.microsoft.com/en-us/onedrive/redirect-known-folders
    • Copy ADMX from %localappdata%\Microsoft\OneDrive\BuildNumber\adm\ to central store
    • Create GPO Workstation - OneDrive for Business and link to Workstations OU
      • “Allow syncing OneDrive accounts for only specific organizations” -> Specify tenant ID
      • “Prevent users from redirecting their Windows known folders to their PC” - Enabled
      • “Silently move Windows known folders to OneDrive” -> Enabled, specify tenant ID, show notification, select all folders
      • “Silently sign in users to the OneDrive sync app with their Windows credentials” -> Enabled
      • “Specify the OneDrive location in a hybrid environment” -> Enabled, set to SharePoint Online
  2. Configure Enterprise State Roaming (ESR)
    • https://docs.microsoft.com/en-us/azure/active-directory/devices/enterprise-state-roaming-enable
