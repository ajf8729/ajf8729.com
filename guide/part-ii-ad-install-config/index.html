Part II: Active Directory - Installation & Configuration

  1. Install Active Directory
    • https://github.com/ajf8729/Lab/blob/main/02-CreateAD.ps1
  2. Configure Active Directory
    • https://github.com/ajf8729/Lab/blob/main/03-ConfigureAD.ps1
  3. Import security baselines
    • https://aka.ms/securitybaselines
    • Copy-Item -Path “C:\LAB\Windows-10-Windows Server-v20H2-Security-Baseline-FINAL\Templates*” -Destination “C:\Windows\SYSVOL\domain\Policies\PolicyDefinitions" -Recurse -Force
    • Copy-Item -Path “C:\LAB\Windows-10-v21H1-Security-Baseline-FINAL\Templates*” -Destination “C:\Windows\SYSVOL\domain\Policies\PolicyDefinitions" -Recurse -Force
    • C:\LAB\Windows-10-v21H1-Security-Baseline-FINAL\Scripts\Baseline-ADImport.ps1
    • C:\LAB\Windows-10-Windows Server-v20H2-Security-Baseline-FINAL\Scripts\Baseline-ADImport.ps1
    • https://github.com/ajf8729/Lab/blob/main/03a-SecurityBaselines.ps1
  4. Configure GPOs
    • Server - ConfigMgr
      • User Rights Assignments
        • “Lock pages in memory”: LAB\svc_CM_SQL$
        • “Log on as a service”: LAB\svc_CM_SQL$
      • Windows Firewall -> New inbound rule
        • Custom
        • All Programs
        • Local TCP ports 1433, 4022
        • Any:Any, Allow, All profiles
        • Name: “LAB - ConfigMgr - SQL”
    • Server - Default Security Policy -> New local group preference
      • Add LocalAdmin_Servers to Administrators
      • Set group description to “GPO-Enforced Group”
      • Enable “Delete all member users” and “Delete all member groups”
    • Workstation - Default Security Policy -> New local group preference
      • Add LocalAdmin_Workstations to Administrators
      • Set group description to “GPO-Enforced Group”
      • Enable “Delete all member users” and “Delete all member groups”
