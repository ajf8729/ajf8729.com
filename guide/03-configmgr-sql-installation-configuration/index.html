Part 03: ConfigMgr - SQL Installation and Configuration

  1. Join ConfigMgr server to domain
    • https://github.com/ajf8729/Lab/blob/main/DomainJoinLabVM.ps1
# Example syntax to join domain using DomainJoinLabVM.ps1
# Run from host running Hyper-V
# This script will prompt for appropriate domain credentials

$Parameters = @{
    LabName    = LAB01
    VMHostname = LAB01DC01A
    Username   = Administrator
    Password   = '3d05c285-f4cb-49bc-bc0f-9ee196a31339'
    DomainName = lab01.domain.tld
    Verbose    = $true
}

.\DomainJoinLabVM.ps1 @Parameters
  1. Configure ConfigMgr Server
    • https://github.com/ajf8729/Lab/blob/main/ConfigureConfigMgrServer.ps1
# Example syntax to configure ConfigMgr Server using ConfigureConfigMgrServer.ps1
# Run from within your ConfigMgr VM
# This script requires no input; it will automatically configure disks, install RSAT tools, install gMSA, and restart

.\ConfigureConfigMgrServer.ps1
  1. Stage media on ConfigMgr server
  • Windows 11 ADK
    • https://docs.microsoft.com/windows-hardware/get-started/adk-install#download-the-adk-for-windows-11
    • https://download.microsoft.com/download/1/f/d/1fd2291e-c0e9-4ae0-beae-fbbe0fe41a5a/adk/adksetup.exe
    • https://download.microsoft.com/download/5/5/e/55e3e34a-5708-46cd-a90d-92044c29336b/adkwinpeaddons/adkwinpesetup.exe
    • adksetup.exe /layout F:\SOURCE\ADK\ADK
    • adkwinpesetup.exe /layout F:\SOURCE\ADK\WinPE
  • ConfigMgr Current Branch Evaluation Media (2103 as of 2022-02-07)
    • https://www.microsoft.com/en-us/evalcenter/evaluate-microsoft-endpoint-configuration-manager
    • https://download.microsoft.com/download/8/8/8/888d525d-5523-46ba-aca8-4709f54affa8/MEM_Configmgr_2103.exe
  • SQL Server 2019 Developer Media
    • https://www.microsoft.com/sql-server/sql-server-downloads
    • https://go.microsoft.com/fwlink/?linkid=866662
  • SQL Server 2019 installation configuration file
    • https://github.com/ajf8729/Lab/blob/main/conf/SQLConfig.ini
    • https://raw.githubusercontent.com/ajf8729/Lab/main/conf/SQLConfig.ini
  • SQL Server 2019 LCU (CU 15 as of 2022-02-07)
    • https://www.microsoft.com/en-us/download/details.aspx?id=100809
    • https://download.microsoft.com/download/6/e/7/6e72dddf-dfa4-4889-bc3d-e5d3a0fd11ce/SQLServer2019-KB5008996-x64.exe
  • SQL Server Management Studio (18.10 as of 2022-02-07)
    • https://aka.ms/ssmsfullsetup
  1. Install SQL Server 2019
  • setup.exe /IAcceptSQLServerLicenseTerms /ConfigurationFile="F:\SOURCE\SQLConfig.ini"
  1. Install SQL Server 2019 LCU
  • SQLServer2019-KB5004524-x64.exe /qs /IAcceptSQLServerLicenseTerms /Action=Patch /AllInstances
  1. Install SQL Server Management Studio
  • SSMS-Setup-ENU.exe /install /passive
  1. Restart ConfigMgr server
