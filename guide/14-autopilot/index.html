14 Autopilot

  1. Configure Autopilot profile
    • https://endpoint.microsoft.com/#blade/Microsoft_Intune_Enrollment/DeploymentProfilesBlade
    • Create -> Windows PC
    • Name: “AJF8729 LAB Default Autopilot Profile”
    • Convert all targeted devices to Autopilot -> Enabled

    • Assignments: All devices
